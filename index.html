This is a proof-of-concept for a data visualization on Venezuelan oil subsidies. Specifically, it compares Dept-to-GSP ratio and Annual Financing as a percent of GDP (oil subsidies from Venezuela). It accomplishes this using an interactive map with a bivariate classification scheme.
